# Git Scripts
The **GitScripts** *(gits)* is a simple bash script with aliases of `git` commands.

# How to install
1. `git clone https://github.com/adisteyf/gits`.
2. `chmod +x ./gits/gits`.
3. `sudo mv ./gits/gits /usr/bin/`.
4. `rm -rf ./gits/`.

# How to delete
1. `sudo rm -f /usr/bin/gits`.

# How to use
`gits -help`.
